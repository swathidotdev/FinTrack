# Budget Tracker

A simple Flask-based personal finance tracker with SQLite persistence, transaction management, CSV import, and budget summaries.

## Features
- Add and view transactions
- Categorize expenses and income
- View totals and balance
- Import transactions from CSV
- Simple dashboard UI

## Run locally
1. Install dependencies: `pip install -r requirements.txt`
2. Start the app: `python project.py`
3. Open http://127.0.0.1:5000
